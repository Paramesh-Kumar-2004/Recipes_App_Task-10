# Recipes App

## Description

A simple application for browsing, searching, and managing recipes.

## Features

- Create Recipe
- Get All Recipe
- Get Recipe By Id
- Delete Recipe
- Update Recipe

## Technologies

- Node Js
- Express Js
- MongoDB
